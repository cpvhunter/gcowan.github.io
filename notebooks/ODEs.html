Solving first order ordinary differential equations (ODEs)¶

A general first order ODE can be expressed as \(\frac{dy(x)}{dx} = f(x, y)\). For example: \(\frac{dy(x)}{dx} = y(x) + x + 2x^2\). Typically, a physics problem may involve higher-order differential equations, but these can always be reduced to studying a set of coupled first order equations, i.e., \(\frac{dy_i(x)}{dx} = f_i(x, y_0, \dots, y_{N-1})\), where the functions \(f_i\) on the RHS are known.

Solving the ODE for the form of the \(y_i(x)\) functions may be possible analytically. However, where this is not possible, numerical methods can be used. In this example we use some numerical methods to solve first order ODEs, showing some differences in accuracy of the approaches. Knowledge of the ODE iteself is not sufficient to solve it, we must also know the boundary conditions, which define that value of the functions at specific points.

def euler(f, x0, y0, steps, xmin, xmax):
    """Implements the Euler method for solving an ODE
    for the function y between xmin and xmax,
    given initial conditions y(x0) = y0
    """
    x = [x0]
    y = [y0]
    delta = float(xmax - xmin)/steps
    for i in range(1, steps+1):
        y.append(y[-1] + delta*f(x[-1] + delta, y[-1]))
        x.append(x[-1] + delta)
    return x, y

It's clear that for too small steps, it becomes difficult to follow the exact form of the curve. Now let's implement the midpoint Runge-Kutta method (or second order RK), where we take the gradient to the curve at the midpoint of range in each iteration and then use that gradient when extrapolating from the start of the range.

def RK2(f, x0, y0, steps, xmin, xmax):
    """Implements the midpoint RK method for solving an ODE
    for the function y between xmin and xmax,
    given initial conditions y(x0) = y0
    """
    x = [x0]
    y = [y0]
    y_mid = 0.
    delta = float(xmax - xmin)/steps
    for i in range(1, steps+1):
        k1 = delta*f(x[-1], y[-1])
        k2 = delta*f(x[-1] + delta/2., y[-1] + k1/2.)
        y.append(y[-1] + k2)
        x.append(x[-1] + delta)
    return x, y

def RK4(f, x0, y0, steps, xmin, xmax):
    """Implements the 4th order RK method for solving an ODE
    for the function y between xmin and xmax,
    given initial conditions y(x0) = y0
    """
    x = [x0]
    y = [y0]
    y_mid = 0.
    delta = float(xmax - xmin)/steps
    for i in range(1, steps+1):
        k1 = delta*f(x[-1]           , y[-1])
        k2 = delta*f(x[-1] + delta/2., y[-1] + k1/2.)
        k3 = delta*f(x[-1] + delta/2., y[-1] + k2/2.)
        k4 = delta*f(x[-1] + delta   , y[-1] + k3)
        y.append(y[-1] + k1/6. + k2/3. + k3/3. + k4/6.)
        x.append(x[-1] + delta)
    return x, y
